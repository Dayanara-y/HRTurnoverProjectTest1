This contains the Java coding for the HR Turnover project. 
requirements:
java 17 or java 17 and higher, intelliJ IDEA, MySQL Worknbench, and MySQL server running on local host:3306
Database setup:
open MySQL Workbench, import data, make sure password, URL, USER info match to the ones in the ConnToDB.java file.
Running the Database:
Open IntelliJ IDEA, Ensure MAVEN is downloaded, run the LoginUI.java file. Enter login credentials based on which dashboard you wish to log into.
